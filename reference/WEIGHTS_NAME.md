# Weight file names in HUB

Weight file names in HUB

## Usage

``` r
WEIGHTS_NAME()

WEIGHTS_INDEX_NAME()
```

## Value

A string with the default file names for indexes in the Hugging Face
Hub.

## Functions

- `WEIGHTS_NAME()`: Name of weights file

- `WEIGHTS_INDEX_NAME()`: Name of weights index file

## Examples

``` r
WEIGHTS_NAME()
#> [1] "pytorch_model.bin"
WEIGHTS_INDEX_NAME()
#> [1] "pytorch_model.bin.index.json"
```
